## 🏁 Run It

No tooling required — just open the HTML file:

```bash
git clone https://github.com/Shashu1007/my-porfile.git
cd my-porfile
# macOS
open frst.html
# Linux
xdg-open frst.html
# Windows
start frst.html
```

## 🪞 Reflection

Looking back at this is fun:

- ✅ Pure HTML — no frameworks
- ✅ Inline styling — long before I discovered Tailwind 😅

It’s a reminder that **shipping > perfection** when you’re learning.
